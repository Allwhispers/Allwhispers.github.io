# Js中的基本数据类型

基本类型分为以下六种：

  • string（字符串）
  • boolean（布尔值）
  • number（数字）
  • symbol（符号）
  • null（空值）
  • undefined（未定义）

关于symbol

symbol是ES6中新增的数据类型，symbol 表示独一无二的值，通过 Symbol 函数调用生成，由于生成的 symbol 值为原始类型，所以 Symbol 函数不能使用new 调用；

关于null 和 undefined

null 和 undefined 通常被认为是特殊值，这两种类型的值唯一，就是其本身

至于这些数据类型主要有哪些特征就不再赘述了，主要记录一下如何判断js中的基本数据类型和js的类型转换（马上想学习下ts，就用这篇博客来为自己几年来无休止的js数据类型困扰划上休止符吧）

# 1.2 instaceof操作符来判断数据类型

一般来说，判断对象数据类型采用的是instanceof，基本原理就是判断实例的隐式原型（proto）是否在构造函数的原型链上 其基本功能的手动实现网上一般是这个代码

function instance_of(L, R) {//L 表示左表达式，R 表示右表达式
        var O = R.prototype;// 取 R 的显示原型
        L = L.__proto__;// 取 L 的隐式原型
        while (true) {
            if (L === null)
                return false;
            if (O === L)// 这里重点：当 O 严格等于 L 时，返回 true
                return true;
            L = L.__proto__;
        }
    }

L instanceof R底层的运算机制关键点如下：

  1. R的数据类型必须为[object Function]，否则就抛TypeError；

而且instance有一个更大的问题是不能确定唯一性 例如

arr instanceof Object   // true

# 1.3终极解决方案Object.toString().call()

function typeOf(obj) {
  const toString = Object.prototype.toString;
  const map = {
    '[object Boolean]'  : 'boolean',
    '[object Number]'   : 'number',
    '[object String]'   : 'string',
    '[object Function]' : 'function',
    '[object Array]'    : 'array',
    '[object Date]'     : 'date',
    '[object RegExp]'   : 'regExp',
    '[object Undefined]': 'undefined',
    '[object Null]'     : 'null',
    '[object Object]'   : 'object'
  };
  return map[toString.call(obj)];
}

# 2 js中的类型转换规则

# 2.1 toString()方法

首先我们来看一下Object.prototype.toString()方法
toString()方法返回一个表示该对象的字符串。
每个对象都有一个 toString() 方法，当然也可以自己重写toString()方法，当对象被表示为文本值时或者当以期望字符串的方式引用对象时，该方法被自动调用。

未被重写的toString()方法

  • Undefined，null，boolean,直接加上引号，例如'null'
  • number 则有比较长的规范,毕竟范围比较大, 常见的就是 '1' NaN则为'NaN' 基本等同于上面一条,对于负数，则返回-+字符串 例如 '-2' 其他的先不考虑了。
  • 对象则是先转为原始值，再按照上面的步骤进行处理。

# 2.2 valueOf()方法

valueOf 当调用 valueOf 方法，采用如下步骤：

  • 调用ToObject方法得到一个对象O
  • 原始数据类型转换为对应的内置对象， 引用类型则不变
  • 调用该对象(O)内置valueOf方法.

不同内置对象的valueOf实现：

  • String => 返回字符串值
  • Number => 返回数字值
  • Date => 返回一个数字，即时间值,字符串中内容是依赖于具体实现的
  • Boolean => 返回Boolean的this值
  • Object => 返回this
